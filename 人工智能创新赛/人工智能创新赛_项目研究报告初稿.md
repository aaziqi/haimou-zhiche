# 海眸智澈：面向水下机器人视觉任务的智能图像增强与感知验证系统
# 项目研究报告初稿

## 摘要

水下图像在采集过程中容易受到光吸收、散射、悬浮颗粒和非均匀光照等因素影响，普遍存在偏色、低对比、模糊和细节丢失问题，严重制约水下机器人巡检、海洋观测和目标识别等视觉任务的可靠性。针对这一问题，本项目研究并实现了一套面向水下机器人视觉任务的智能图像增强与感知验证系统。项目以改进 CycleGAN 为核心增强模型，在无配对样本条件下，引入颜色约束、结构一致性约束和感知损失，提升水下图像增强质量；同时集成传统增强方法、自动指标评测、下游检测对照、报告生成和案例归档等功能，形成一套完整的软件化平台。实验与系统应用结果表明，该项目不仅能够改善水下图像视觉效果，还能够为后续视觉感知任务提供支持，具有较强的研究价值、工程意义和竞赛展示价值。

**关键词：** 水下图像增强；CycleGAN；目标检测；机器人视觉；交互式评测系统

## 第一章 项目背景与研究意义

随着水下机器人在海洋资源勘测、生态监测、水下工程巡检、搜救和国防等领域的广泛应用，水下视觉感知能力的重要性不断提升。然而，真实水下环境具有光衰减强、颜色失真明显、悬浮颗粒多、成像噪声大等特点，导致采集图像常常出现蓝绿偏色、对比度不足和细节缺失问题。这些问题不仅影响人工观察，还会降低目标检测、识别和定位等后续视觉算法的效果。

传统的图像增强方法如白平衡、直方图均衡化、CLAHE 和 Gamma 校正等，虽然具有实现简单、计算开销小的特点，但在复杂水下环境中往往难以兼顾颜色恢复、结构保持和纹理清晰度。随着深度学习的发展，基于 GAN 的图像增强方法逐渐成为研究热点，其中 CycleGAN 因能够在无配对数据条件下完成图像域转换而受到广泛关注。

但是，现有很多研究更注重算法精度或主观视觉效果，常常停留在论文实验层面，缺少适合竞赛展示和工程验证的系统化实现。针对这一问题，本项目希望实现两个目标：一是在算法层面对水下图像增强模型进行改进；二是在系统层面将增强、评测、展示和下游验证整合为一个完整的软件平台。该项目既具有理论研究意义，也具有明显的工程应用和竞赛落地价值。

## 第二章 国内外研究现状

### 2.1 传统水下图像增强研究

传统水下图像增强方法主要包括白平衡、直方图均衡化、Retinex、Gamma 校正、对比度拉伸和基于物理模型的去雾方法等。这类方法通常计算效率高、实现简单、可解释性强，适用于对图像进行快速预处理。但是，由于水下成像退化机理复杂，传统方法在不同场景下稳定性不足，常出现颜色过补偿、对比度异常或者局部细节损伤等问题。

### 2.2 深度学习水下图像增强研究

随着卷积神经网络和生成对抗网络的发展，越来越多研究开始采用深度学习方法处理水下图像增强任务。相比传统方法，深度学习方法能够通过大规模数据学习复杂退化特征与清晰图像之间的映射关系，通常在主观视觉和客观指标上都具有更好的表现。

其中，GAN 类方法通过生成器和判别器的博弈学习清晰图像分布，能够更好地恢复视觉自然度。CycleGAN 进一步解决了增强任务对严格配对数据的依赖问题，使其在真实水下场景中具有更强适应性。因此，基于 CycleGAN 的改进方法成为水下图像增强的一个重要研究方向。

### 2.3 现有工作的不足

尽管已有研究在模型设计和实验指标方面取得一定成果，但仍存在以下不足：

1. 很多工作关注算法精度，缺少面向实际应用的系统集成。
2. 很多研究仅比较增强前后视觉效果，缺少与下游任务联动的验证。
3. 很多公开成果停留在代码和论文层面，缺少适合比赛答辩和工程展示的人机交互系统。

因此，构建一个兼顾算法改进、系统实现和应用验证的项目，具有较强的研究价值和现实意义。

## 第三章 项目总体设计

本项目整体采用“前端展示层 + Python 服务层 + 模型推理层 + 结果归档层”的系统架构。

### 3.1 前端展示层

前端部分负责用户交互和展示，包括首页、智能增强页、方案对比页、指标图表页、结果归档页和演示流程页。前端不仅展示图像结果，还负责显示指标、报告链接、案例封面和系统亮点，使作品具备完整的软件产品特征。

### 3.2 Python 服务层

服务层负责接收前端请求，并完成模型调用、传统方法执行、指标计算、结果文件生成和归档信息写入等功能。服务接口包括：

- `/api/bootstrap`
- `/api/enhance`
- `/api/compare`
- `/api/detect_compare`
- `/api/archive`

这些接口使前端与算法推理逻辑有效解耦，便于系统扩展和部署。

### 3.3 模型推理层

模型推理层主要包括三类方法：

1. 传统方法：Gray World、CLAHE、Gray World + CLAHE、Gamma
2. 深度学习基线方法：原始 CycleGAN
3. 改进模型：MP-CycleGAN

该层负责执行图像增强和相关中间结果输出，为评测和展示提供基础数据。

### 3.4 结果归档层

系统采用文件化方式对结果进行组织，包括 HTML 报告、CSV 指标、JSON 检测结果和归档索引文件。相较传统数据库，文件化方案更加轻量、便于迁移，也更适合竞赛环境中的本地演示和提交整理。

## 第四章 关键算法与关键技术

### 4.1 基于 CycleGAN 的无配对水下图像增强

CycleGAN 的核心思想是在两个图像域之间建立双向映射，通过对抗损失和循环一致性损失，在无配对样本条件下学习图像转换关系。本项目将其用于“退化水下图像域”到“增强图像域”的映射学习，从而避免对严格配对数据的高度依赖。

该方法的优势在于：

- 适应真实场景数据获取困难的问题；
- 能够学习较复杂的颜色和纹理分布变换；
- 适合水下图像增强这类难以构建精确配对样本的任务。

### 4.2 改进模型 MP-CycleGAN

为了更贴合水下图像退化特性，本项目在标准 CycleGAN 基础上设计了改进模型 MP-CycleGAN，核心改进如下：

#### （1）颜色约束

针对水下图像常见的蓝绿偏色问题，引入颜色一致性或灰世界相关约束，提升生成结果的色彩平衡能力，减少颜色失真。

#### （2）结构一致性约束

为避免增强过程中出现结构漂移和纹理模糊，引入基于边缘或梯度信息的结构一致性约束，使图像在增强后仍保持主要轮廓和几何结构。

#### （3）感知损失

利用预训练特征网络对高层语义和纹理信息进行约束，使增强结果不仅在像素层面变化，更在视觉主观质量上得到改善。

### 4.3 传统增强方法基线

项目保留传统增强方法作为基线，包括 Gray World、CLAHE、Gray World + CLAHE 和 Gamma 校正。保留这些方法的意义在于：

- 形成清晰的技术演进链条；
- 提高答辩时的可解释性；
- 验证深度模型在复杂场景下的必要性。

### 4.4 多指标评测技术

为了客观评估增强效果，本项目设计了多指标联合评测机制，主要使用：

- UCIQE：评价水下图像颜色和清晰度质量
- UIQM：评价水下图像整体视觉质量
- PSNR：评价增强图像与参考图像之间的像素保真度
- SSIM：评价图像结构相似性

这种设计兼顾无参考和有参考两类评价方式，使结果更加全面。

### 4.5 下游检测联动验证技术

项目引入基于 YOLO 的目标检测模块，对原图和增强图进行检测对照。该设计不再仅停留在“图像更清晰”的层面，而是进一步证明增强对后续视觉感知任务的实际价值。

## 第五章 系统实现与功能模块

### 5.1 首页模块

首页主要展示项目定位、系统亮点、主推模型、数据摘要和导航入口，作用是帮助评委和用户快速理解作品背景、技术路线和展示重点。

### 5.2 智能增强模块

用户可上传一张水下图片，选择增强模型并执行单图增强。系统将自动输出增强结果、前后对比效果和相关评测结果。

### 5.3 综合对比模块

该模块将传统方法、原始 CycleGAN 和改进模型置于同一页面中进行横向对比，便于直观展示不同方案的差异。

### 5.4 指标图表模块

图表模块将不同方法在 EUVP、UIEB 等数据集上的关键指标以图形方式呈现，帮助用户快速理解实验结果。

### 5.5 结果归档模块

系统会自动记录每次增强、对比和检测任务的输出文件、报告链接和摘要信息，形成可重复查看和复用的案例库。

### 5.6 一键演示流程模块

演示流程页将首页、增强、对比、图表和归档串联起来，方便比赛现场按照固定节奏进行讲解和展示。

## 第六章 实验设计与结果分析

### 6.1 实验目标

本项目实验主要从三个层面展开：

1. 验证改进模型在水下图像增强任务中的效果；
2. 验证系统是否能够稳定支持增强、对比、评测和归档功能；
3. 验证增强结果对下游检测任务的辅助价值。

### 6.2 实验数据与方法

本项目主要使用 EUVP 和 UIEB 数据集开展实验与评测。其中，EUVP 用于训练与主测试，UIEB 用于跨数据集泛化分析。实验中选取传统方法、原始 CycleGAN 和改进模型进行对比，分析其在多个指标上的表现。

### 6.3 代表性实验结果

根据已有实验结果，项目在 EUVP 数据集上的改进模型表现优于基线模型，在 SSIM 和 UIQM 等综合指标上具有一定优势；在 UIEB 数据集上，模型保持了较好的泛化能力，同时也暴露了跨域性能仍有进一步提升空间。

从系统层面看，前端能够稳定加载模型信息、展示图像对比结果并打开报告文件；后端能够完成增强推理、指标计算、检测对照和归档写入，整体链路较为完整。

从应用层面看，通过将增强结果接入检测任务，可以观察到图像视觉增强对目标轮廓辨识和检测可视化具有一定帮助，这说明图像增强对实际机器人视觉任务具有潜在支撑作用。

## 第七章 创新点与应用价值

### 7.1 项目创新点

1. 提出面向水下场景的改进增强模型，在无配对条件下融合颜色、结构和感知多重约束。
2. 将图像增强与下游检测联动，形成“增强 - 检测”应用验证链。
3. 将传统方法、深度模型、评测指标、自动报告和案例归档整合为统一平台。
4. 支持本地动态推理与静态展示双模式，提升竞赛答辩和成果传播能力。

### 7.2 应用价值

本项目可应用于以下场景：

- 水下机器人巡检前的图像预处理
- 海洋生态监测图像增强辅助
- 水下目标识别与检测前处理
- 教学实验与科研演示平台
- 竞赛答辩和成果展示系统

相较于单纯算法模型，本项目以软件系统形式输出成果，更易被实际应用和扩展。

## 第八章 不足与后续改进

虽然本项目已经完成了从算法到系统的基本落地，但仍存在以下不足：

1. 在跨数据集泛化能力方面，模型仍有进一步优化空间。
2. 当前系统更偏向本地单机演示，若面向公网部署仍需进一步改造服务架构。
3. 下游任务目前以目标检测为主，后续可扩展到分割、识别和测量等任务。
4. 当前案例库和数据规模仍可继续扩充，以提升系统的展示丰富度和研究支撑能力。

后续计划包括：

- 引入更稳定的跨域增强策略
- 扩展更多数据集与基线模型
- 增加更多下游感知任务
- 优化线上展示版本和轻量化部署方案

## 第九章 结论

本项目围绕水下机器人视觉感知中的图像退化问题，提出并实现了一套集算法改进、评测验证和系统展示于一体的智能图像增强与感知验证系统。项目以改进 CycleGAN 为核心，通过颜色、结构和感知多重约束提升增强质量；同时通过传统方法对照、客观指标评测、下游检测验证和自动归档机制，构建了完整的软件化应用链条。研究与实现结果表明，本项目不仅具有一定的技术创新性，也具备较强的竞赛适配性和工程展示价值。
