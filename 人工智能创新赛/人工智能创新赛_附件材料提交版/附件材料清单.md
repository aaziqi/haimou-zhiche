# 人工智能创新赛 附件材料清单

## 一、附件材料目录说明

本文件夹用于配合“中国机器人及人工智能大赛 人工智能创新赛”报名与材料提交，已按申报习惯整理为以下 7 类内容：

1. `01_申报文稿`
2. `02_项目研究原始资料`
3. `03_系统界面截图`
4. `04_结构图与实验图`
5. `05_代表性结果`
6. `06_文献检索附件`
7. `07_活动照片与日志`

## 二、各目录内容

### 1. 申报文稿

- 报名表最终版文案
- 查新报告定稿
- 项目研究报告定稿
- 报名表与查新报告模板

### 2. 项目研究原始资料

- 关键算法与技术创新整理文档
- 模型结构图
- 指标雷达图
- 训练损失曲线
- 方法对比图

说明：

- 该目录用于支撑“项目研究原始资料（图纸、图表等）”栏目。

### 3. 系统界面截图

- 首页截图
- 图表页截图
- 归档页截图
- 演示流程页截图
- 演示视频封面
- LOGO

说明：

- 该目录用于支撑“项目研究活动照片/系统界面展示/附件图片材料”等栏目。

### 4. 结构图与实验图

- 模型结构图
- 指标雷达图
- 训练损失曲线
- 方法对比图

说明：

- 该目录便于单独引用到研究报告、查新报告和答辩 PPT 中。

### 6. 文献检索附件

- 文献检索附件摘要整理

说明：

- 该目录用于支撑查新报告中的“附件文献摘要”部分。

### 7. 活动照片与日志

- 活动照片说明
- 活动日志说明

说明：

- 当前目录内为占位说明文件。
- 若后续补充项目活动照片、指导交流记录、实验记录等材料，可直接替换或追加到本目录。

## 三、报名表填写建议

若按当前附件材料整理情况填写报名表中的“附件材料”栏目，可参考：

- 项目研究原始资料：有
- 项目研究活动照片：如暂未补充，可填“无”或“若干页（系统界面截图代替展示）”
- 项目研究活动日志：如暂无完整记录，可填“无”
- 其他：系统界面截图、结构图、实验图、代表性增强结果、检测对照结果、文献检索摘要

## 四、备注

本附件材料目录以“方便提交、方便查找、方便答辩展示”为原则整理。后续如你补充队伍合照、项目开发过程照片、指导记录、原始实验表格等内容，可继续并入本目录，不影响现有结构。
